## Findings (Credibility Gaps)
1) Conclusive framing appears before evidentiary grounding in some headings.
2) Hero metrics lack visible primary sources on the same block.
3) Timeline mixes "verified fact" with "evidence pending" without explaining the distinction.
4) Evidence tiers (official vs media vs citizen data) are not labeled or ranked.
5) No visible correction or methodology policy, which weakens journalistic defensibility.
6) Some claims are vulnerable to counter-argument unless marked as reported vs observed.

## Trust Signals to Implement
- Source boxes per section: "Fuente primaria", "Archivo", "Ultima verificacion".
- Evidence status tags: verificado / pendiente / en disputa.
- Versioning: visible "last verified" and update log.
- Methodology + corrections page.
- Source tier labels (official, observer, NGO, media, citizen).

## Metodo y Correcciones (Draft)
### Metodo (resumen corto)
Este observatorio documenta incumplimientos verificables del CNE con un criterio minimo: cada afirmacion debe tener
una fuente primaria o un archivo verificable. Cuando no existe evidencia primaria, el hecho se marca como "pendiente"
o "en disputa" y se explica la razon.

### Niveles de evidencia
- Oficial: CNE, TSJ, Gaceta, organismos internacionales.
- Observadores: misiones y paneles con metodologia publicada.
- ONG: informes con metodologia verificable.
- Media: cobertura periodistica con fuentes identificables.
- Ciudadana: actas y bases de datos verificables.

### Registro de actualizaciones
Se mantiene un registro publico con la fecha de verificacion, cambios en el estado de evidencia y nuevas fuentes.

### Correcciones
Si detectas un error, envia la fuente corregida y una breve explicacion al canal oficial del proyecto.
Las correcciones se registran con fecha, motivo y referencia.
